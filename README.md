# 프로그래밍과 데이터 기초 단위 프로젝트

## 프로젝트 주제

- 전국 전기자동차 등록 현황 및 자동차 기업 FAQ 조회 시스템
    - 시도별 전기차 등록대수·인구당 보유율, 충전소 비율, 보조금을 다각도로 비교
    - 자동차 기업별 FAQ 및 A/S·정비소 정보 검색 제공

---

## 프로젝트 목표

- 업무 도메인에 필요한 데이터 정의
- 파이썬 기반 Beautiful Soup과 Selenium 활용 데이터 수집
- MySQL DB 설계 및 데이터 저장
- 파이썬과 MySQL 연동 조회 시스템 개발
- Streamlit 활용 데이터 조회 GUI 구현

---

## 프로젝트 내용

- 데이터 조회에 사용할 데이터 정의
- 공공데이터포털 API 및 웹 크롤링을 통한 데이터 수집
- MySQL 설치 및 환경 설정
- MySQL ERD 설계 및 수집한 데이터 저장
- 파이썬 활용 MySQL 연동하여 SQL로 데이터 조회
- Streamlit 활용 데이터 조회 GUI 구현

---

## 필수 산출물 (Github 업로드)

- 데이터베이스 설계 문서 (ERD)
- 데이터 수집 프로그램 (MySQL)
- 데이터 조회 프로그램 (Streamlit)

## 주요 기능

- 지역별 전기차 등록대수 조회 및 인구당 보유율 비교 (참고 지표)
- 지역별 전기차 대비 충전소 비율 비교 (전기차 1대당 충전기 수)
- 지역별 + 차종별 전기차 보조금(국비+지방비) 비교
- 자동차 기업 FAQ 검색 및 조회
    - 기업별 전기차 A/S 관련 정보
    - 기업별 정비소 위치 정보
- (선택) 차종별 차량 가격 비교 (현대·기아 한정)

## 분석 기준

| 지표 | 계산식 | 비고 |
|---|---|---|
| 인구당 보유율 | 전기차 등록대수 ÷ 시도별 인구수 × 1,000 | 천명당 전기차 수 |
| 충전소 비율 | 충전기 수 ÷ 전기차 등록대수 | 전기차 1대당 충전기 수 |
| 보조금 합계 | 국비 + 지방비 | 차종별·지역별 |

## 구현 가능성 검토

### 가능한 부분

| 항목 | 방법 | 난이도 |
|---|---|---|
| 시도별 전기차 등록대수 | 국토교통부 공공데이터포털 CSV / API | 하 |
| 시도별 인구수 | 행정안전부 주민등록인구 / KOSIS API | 하 |
| 시도별 충전소 수 | 한국전력공사 / 한국환경공단 공공데이터포털 API | 하 |
| 차종별 보조금 | 무공해차 통합누리집(ev.or.kr) 크롤링 | 중 |
| 자동차 기업 FAQ | 제조사 홈페이지 BeautifulSoup 크롤링 | 중 |
| 정비소 위치 | 현대·기아 홈페이지 크롤링 | 중 |
| MySQL 연동 | pymysql / mysql-connector-python | 하 |
| Streamlit GUI | 공식 라이브러리 활용 | 하 |

### 어려운 부분 및 제약

- **보조금 데이터**: 연도·지역·차종별로 금액이 상이하며 매년 갱신됨 → 수집 기준 연도 명시 필수
- **충전소 데이터**: 한국전력공사·환경부 등 출처가 복수 → 단일 출처로 통일 필요
- **정비소 데이터**: 공식 API 없음 → 홈페이지 크롤링, robots.txt 사전 확인 필수
- **차량 가격 (선택)**: 공식 홈페이지 SPA 구조 → Selenium 필요
