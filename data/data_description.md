# 데이터 목록 및 수집 방법

> 비교 기준: 절대 등록대수 / 인구당 보유율(등록대수 ÷ 인구수) / 전년 대비 증가율((당해 − 전년) ÷ 전년 × 100)

---

## 수집 우선순위 요약

| 우선순위 | 데이터 | 공개 여부 | 방법 | 난이도 |
|---|---|---|---|---|
| 1 | 시도별 전기차 등록 현황 | ✅ 공개 | 공공데이터포털 CSV / API | 하 |
| 2 | 시도별 인구수 | ✅ 공개 | 통계청 KOSIS API | 하 |
| 3 | 자동차 기업 FAQ | ⚠️ 크롤링 | 제조사 홈페이지 BeautifulSoup | 중 |
| 4 | 유가 데이터 (선택) | ✅ 공개 | 오피넷 API | 하 |

---

## 1. 시도별 전기차 등록 현황

**용도**: 지역별 절대 등록대수 조회 및 전년 대비 증가율 계산 기준값

| 항목 | 내용 |
|---|---|
| 추천 방법 | **국토교통부 자동차 등록통계** 공공데이터포털 CSV 다운로드 또는 API 조회<br>→ 시도별·연도별 전기차 등록대수가 정리돼 있어 바로 활용 가능 |
| 출처 | [국토교통부 자동차 등록 현황 (공공데이터포털)](https://www.data.go.kr/data/15109006/fileData.do) |
| 필요 항목 | 연도, 시도명, 전기차 누적 등록대수 |
| 수집 주기 | 연 1회 (분석 대상 연도 범위 확정 후 일괄 수집) |
| 주의 | 개인·법인 등록 합산 여부 확인 필요. 분석 연도 범위(3~5개년)는 팀 협의 후 확정 |

---

## 2. 시도별 인구수

**용도**: 인구당 전기차 보유율 계산 기준값

| 항목 | 내용 |
|---|---|
| 추천 방법 | **통계청 국가통계포털(KOSIS)** API 또는 CSV 다운로드<br>→ 행정구역별 연도별 인구수 무료 제공 |
| 출처 | [통계청 KOSIS 행정구역별 인구 현황](https://kosis.kr/statHtml/statHtml.do?orgId=101&tblId=DT_1B040A3) |
| 필요 항목 | 연도, 시도명, 인구수 |
| 수집 주기 | 연 1회 (전기차 등록 데이터와 동일 연도 기준) |
| 주의 | 전기차 등록 데이터와 연도·지역 단위(17개 시/도) 일치 여부 확인 필요 |

---

## 3. 자동차 기업 FAQ

**용도**: 자동차 기업별 FAQ 검색 및 조회 기능 제공

| 항목 | 내용 |
|---|---|
| 추천 방법 | 대상 제조사 공식 홈페이지 FAQ 페이지 **BeautifulSoup** 크롤링<br>→ SPA 구조 사이트는 Selenium 병행 |
| 출처 | 대상 기업 공식 홈페이지 FAQ 페이지 (수집 대상 기업 목록은 팀 협의 후 확정) |
| 필요 항목 | 기업명, 질문, 답변, FAQ 카테고리 |
| 수집 주기 | 최초 1회 수집 후 필요 시 업데이트 |
| 주의 | 크롤링 전 robots.txt 및 이용약관 사전 확인 필수. SPA 구조 여부에 따라 Selenium 필요 |

---

## 4. 유가 데이터 (선택)

**용도**: 유가 변동률과 전기차 증가율 간 상관관계 분석

| 항목 | 내용 |
|---|---|
| 추천 방법 | **오피넷 REST API** — 전국 평균 유가 무료 조회 |
| 출처 | [오피넷 유류가격 API](https://www.opinet.co.kr/user/custapi/custApiInfo.do) (한국석유공사) |
| 필요 항목 | 연도(또는 월), 전국 평균 유가 (휘발유 기준) |
| 수집 주기 | 연간 평균 또는 월별 데이터 1회 수집 |
| 주의 | API 키 발급 필요 (무료). 시간 여유가 있을 경우에만 구현 |
