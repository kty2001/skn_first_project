# 데이터 목록 및 수집 방법

> 총비용 = 차량 가격 + 유류비 + 보험료 + 소모품 비용

---

## 수집 우선순위 요약

| 우선순위 | 데이터 | 공개 여부 | 방법 | 난이도 |
|---|---|---|---|---|
| 1 | 연비 | ✅ 공개 | 공공 CSV 다운로드 | 하 |
| 2 | 유류비 시세 | ✅ 공개 | 오피넷 API | 하 |
| 3 | 자동차 모델 목록 | ✅ 공개 | 연비 CSV 필터링 + 공식 사이트 크롤링 | 중 |
| 4 | 신차 가격 | ⚠️ 크롤링 | 다나와 크롤링 | 중 |
| 5 | 소모품 비용 | ❌ 없음 | 고정 테이블 작성 | 하 |
| 6 | 보험료 | ⚠️ 수동 | 수동 수집 후 고정값 | 하 |
| 7 | 중고차 시세 | ⚠️ 크롤링 | 엔카·KB차차차 크롤링 | 중 |
| 8 | 렌트·리스 가격 | ⚠️ 수동 | 렌트사 홈페이지 수동 수집 | 중 |

## 1. 자동차 모델 목록

**용도**: 비교 대상 차량 목록 구성 (현대·기아 승용차)

| 항목 | 내용 |
|---|---|
| 추천 방법 | ① 연비 CSV에서 제조사·차종 필터링 → 전체 모델 초안 추출<br>② 현대·기아 공식 사이트 Selenium 크롤링 → 현재 판매 중 모델 확인<br>③ 두 결과 병합하여 최종 목록 확정 |
| 출처 | [한국에너지공단 자동차 표시연비 정보](https://www.data.go.kr/data/15083023/fileData.do) |
| 필요 항목 | 제조사, 모델명, 차종(승용), 연료 유형, 판매 여부(현행/단종) |
| 수집 주기 | 최초 1회 수집 후 신모델 출시 시 업데이트 |
| 주의 | 연비 CSV에 단종 모델 포함 → 현행 모델 플래그 별도 처리 필요 |

---

## 2. 신차 가격

**용도**: 신차 구입 비용 (트림별 기본가 기준)

| 항목 | 내용 |
|---|---|
| 추천 방법 | **다나와 자동차** BeautifulSoup + Selenium 크롤링<br>→ 트림별 가격이 정리돼 있어 공식 SPA보다 파싱 용이 |
| 출처 | [다나와 신차견적](https://auto.danawa.com/newcar/) / [현대 가격표 PDF](https://www.hyundai.com/kr/ko/e/vehicles/catalog-price-download) / [기아 가격표 PDF](https://www.kia.com/kr/vehicles/catalog-price) |
| 필요 항목 | 모델명, 트림명, 기본 가격, 연료 유형 |
| 수집 주기 | 연 1~2회 (가격 변동 시) |
| 주의 | 공식 홈페이지는 SPA 구조 → Selenium 필수. PDF 파싱은 pdfplumber 활용 가능 |

## 3. 연비

**용도**: 유류비 계산 기준값

| 항목 | 내용 |
|---|---|
| 추천 방법 | **한국에너지공단 연비 CSV 직접 다운로드** — 무료, 로그인 불필요 |
| 출처 | [한국에너지공단 자동차 표시연비 정보 (CSV)](https://www.data.go.kr/data/15083023/fileData.do)<br>[연비 목록 조회 OpenAPI](https://www.data.go.kr/data/15139827/openapi.do) |
| 필요 항목 | 제조사, 모델명, 연료 유형, 복합연비, 도심연비, 고속도로연비, 1회충전 주행거리(전기차) |
| 수집 주기 | 연 1회 (공단 데이터 갱신 주기에 맞춰) |
| 주의 | 약 2,800개 이상 모델 수록 → 현대·기아 필터링 후 사용 |

## 4. 유류비 시세

**용도**: 연간 유류비 계산 기준 단가

| 항목 | 내용 |
|---|---|
| 추천 방법 | **오피넷 REST API** — 전국 평균 단가 무료 실시간 조회 |
| 출처 | [오피넷 유류가격 API](https://www.opinet.co.kr/user/custapi/custApiInfo.do) (한국석유공사) |
| 필요 항목 | 전국 평균 유가 (휘발유·경유·LPG), 조회 기준일 |
| 수집 주기 | 분석 시점 최신값 1회 조회 또는 월별 평균 저장 |
| 주의 | API 키 발급 필요 (무료). 전기차는 한국전력 전기요금 단가 별도 수집 필요 |

## 5. 보험료

**용도**: 연간 보험료 추정

| 항목 | 내용 |
|---|---|
| 추천 방법 | 보험개발원 공시 기준 차량가액 구간별 평균 보험료 테이블 수동 작성<br>조건 고정: 26세 이상, 운전경력 5년, 자차 포함 |
| 출처 | [보험개발원 공시실](https://www.kidi.or.kr) / [금융위원회 자동차보험가입정보 API](https://www.data.go.kr/data/15124891/openapi.do) (가입 통계용) |
| 필요 항목 | 차량가액 구간, 기준 연간 보험료 |
| 수집 주기 | 최초 1회 수집 (연 1회 검토) |
| 주의 | 실제 보험료는 개인 조건에 따라 편차 큼 → 분석 시 고정 조건 명시 필수<br>공개 API는 보험료 금액이 아닌 가입 건수 통계 제공 |

## 6. 중고차 시세

**용도**: 중고차 구입 비용 (연식·주행거리별 평균 시세)

| 항목 | 내용 |
|---|---|
| 추천 방법 | **엔카 또는 KB차차차** BeautifulSoup 크롤링<br>→ 동일 모델 연식별 평균 시세 수집 |
| 출처 | [엔카](https://www.encar.com) / [KB차차차](https://www.kbchachacha.com) |
| 필요 항목 | 모델명, 연식, 주행거리 구간, 평균 매물 가격 |
| 수집 주기 | 수집 시점 1회 고정 (기준일 DB 저장) |
| 주의 | 실시간 변동 데이터 → 수집 기준일 명시 필수. 크롤링 전 robots.txt 확인 필요 |

## 7. 렌트·리스 가격

**용도**: 장기렌트·리스 월 납입금 기준

| 항목 | 내용 |
|---|---|
| 추천 방법 | 롯데렌탈·SK렌터카·현대캐피탈 기준 견적 페이지에서 수동 수집<br>→ 크롤링 구조 복잡하므로 초기에는 수동 권장 |
| 출처 | 롯데렌탈 / SK렌터카 / 현대캐피탈 공식 사이트<br>[전국렌터카업체정보 표준데이터](https://www.data.go.kr/data/15025689/standard.do) (업체 목록용) |
| 필요 항목 | 모델명, 약정 기간(24·36·48개월), 보증금, 월납금, 반납 조건 |
| 수집 주기 | 최초 1회 수집 (분기별 검토) |
| 주의 | 개인 신용·약정 조건에 따라 금액 상이 → 표준 조건(무보증·48개월 등) 고정 명시 |

## 8. 소모품 교체 비용

**용도**: 기간별 소모품 교체 비용 계산

| 항목 | 내용 |
|---|---|
| 추천 방법 | 공개 데이터 없음 → 정비 표준 주기 기반 고정 테이블 직접 작성 |
| 출처 | 제조사 정비 매뉴얼 / 자동차 정비 커뮤니티 평균값 참고 |
| 필요 항목 | 소모품명, 교체 주기(km 또는 년), 교체 비용(공임 포함 평균) |
| 수집 주기 | 최초 1회 작성 |
| 주의 | 전기차는 엔진오일·점화플러그 제외 → 연료 유형별 소모품 목록 분기 처리 필요 |

**기본 소모품 테이블**

| 소모품 | 교체 주기 | 평균 비용 (공임 포함) | 전기차 해당 |
|---|---|---|---|
| 엔진오일 + 필터 | 1만 km / 1년 | 6~12만 원 | ❌ |
| 타이어 (4개) | 4~5만 km | 40~100만 원 | ✅ |
| 브레이크 패드 (전) | 3~4만 km | 10~20만 원 | ✅ (회생제동으로 주기 길어짐) |
| 에어 필터 | 2만 km / 2년 | 2~5만 원 | ❌ |
| 에어컨 필터 | 1만 km / 1년 | 1~3만 원 | ✅ |
| 배터리 (보조/12V) | 3~5년 | 10~20만 원 | ✅ |
| 점화플러그 | 4만 km | 3~8만 원 | ❌ |
| 냉각수 | 4~5년 | 5~10만 원 | ✅ (전기차는 냉각 시스템용) |
